## Setup

## Prepopulate database

-   Creates 50 random users + test0  
    http://localhost:8080/mock/users/50
-   Each user follows maximum of 10 random users  
    http://localhost:8080/mock/following/10
-   Creates 500 tweets  
    http://localhost:8080/mock/tweets/500
